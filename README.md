# Vyavasth landing page

Marketing site for [Vyavasth](https://vyavasth.in): studio and business operations management for photographers and small teams in India. The app is a Next.js front end with animated sections, legal pages, and server routes that forward enquiries and support requests to the Vyavasth onboarding API.

## Tech stack

- [Next.js 16](https://nextjs.org/) (App Router) with [React 19](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/) and [Lucide React](https://lucide.dev/) for UI motion and icons
- [Mongoose](https://mongoosejs.com/) (connection helper and schemas; not wired into the public form routes today)
